## Dashboard app

### System requirements

### Next.js

[Next.js](https://nextjs.org/)

```bash
v14.0.3
```

Node version

```bash
    v20.10.0
```

Yarn package manager

```bash
1.22.21
```

### Set up

Install deps:

```bash
yarn
#or
yarn install
```

Run the development server:

```bash
yarn dev

```

Open [http://localhost:3000/login](http://localhost:3000/login) with your browser to see the
result.
